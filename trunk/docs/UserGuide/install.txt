POKYNY K INSTALACI KOMPONENT

Softwarová část této práce je určená pro operační systém LINUX.
Pro fungování všech softwarových komponent je nutné do systému nainstalovat programové nástroje třetích stran. Tento soubor obsahuje návod, jak tyto programové nástroje získat a nainstalovat.   

________________________________________________________________________________
1) Stažení SVN repository
(obsahuje zdrojové kódy, skripty, data)
- nutné mít nainstalovaného SVN klienta (např. http://subversion.tigris.org/)

příkaz:
svn checkout http://svn.berlios.de/svnroot/repos/czsem/trunk

(V aktuálním adresáři vytvoří adresář "trunk" a do něho stáhne obsah repository)

________________________________________________________________________________
2) Instalace programu Tree Editor TrEd
Program btred (součást editoru TrEd) je nutný pro fungování většiny lingvistických operací

Instrukce pro instalaci včetně instalačního balíčku je k dispozici na adrese:
http://ufal.mff.cuni.cz/~pajas/tred/
Další možnost, jak získat TrEd, je nainstalovat si ho z PDT2.0 CD-ROM. Viz dále.

Po instalaci je ještě nutné přidat adresář "tred" (obsahující skript "btred") do proměnné PATH.

________________________________________________________________________________
3) Instalace Tools for machine annotation (from PDT2.0)
Jedná o skupinu nástrojů, které provádějí lingvistickou analýzu textu až po analytickou rovinu.

Tyto nástroje obsahuje CD-ROM The Prague Dependency Treebank 2.0.
Instrukce, jak získat tento CD-ROM jsou na stránkách  PDT2.0 v sekci "7. Distribution & License".
http://ufal.mff.cuni.cz/pdt2.0/

Instalace Tools for machine annotation je součástí hlavního instalačního procesu PDT2.0.
Viz sekce "8. Installation" na stránkách  PDT2.0 (http://ufal.mff.cuni.cz/pdt2.0/)
Pro otestování diplomové práce stačí spustit custom instalaci a vybrat položky:
26 -Scripts for Format Conversions, with documentation 98.7 kB
27- Machine Annotation Tools (Linux binaries) 155.9 MB
Jako cílový adresář nastavte "parsing_czech/PDT" v adresáři, kam jste stáhli SVN repository (..../trunk/parsing_czech/PDT).

Tyto nástroje fungují pouze na operačních systémech Linux běžících na i386 architektuře.

Za dodržení licenčních podmínek je též možné si CD-ROM PDT2.0 zapůjčit u autora diplomové práce.

________________________________________________________________________________
4) Nástroj pro tektogramatickou analýzu češtiny - Václav Klimeš
Jedná o nástroj, který provádí tektogramatickou lingvistickou anotaci.
Jako vstup akceptuje výstup nástrojů výše (Tools for machine annotation).

Nástroj lze získat přímo od Václava Klimeše.
http://ufal.mff.cuni.cz/~klimes/ (kontaktní informace)

Pro pouhé otestování funkčnosti diplomové práce lze použít kopii tohoto nástroje v SVN repository (v adresáři trunk/parsing_czech/t-analysis).
Instalace této kopie:

Nejprve je nutné stáhnout a nainstalovat fnTBL Toolkit.
http://nlp.cs.jhu.edu/~rflorian/fntbl/index.html
Návod k instalaci je uvnitř staženého archivu v souboru "README".

Dále je nutné nakonfigurovat cesty k důležitým souborům t-analysis.
V souboru "run" nastavte proměnou "A2T" na cestu k adresáři "t-analysis" a proměnnou "FNTBL" na cestu kde je nainstalovaný  fnTBL Toolkit.
V souborech "t-analysis/[1-4]/cfg/tbl.params" a "t-analysis[1-4]/R" změňte vždy na prvním řádku cestu tak, aby odpovídala aktuálnímu umístění souborů.

Zajistěte, aby adresář s programem btred, což je součást TrEdu, byl v PATH. (Stačí, když na něj bude symlink v nějakém vašem adresáři bin.)
