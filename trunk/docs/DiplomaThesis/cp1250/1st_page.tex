\noindent Abstract: This work starts with the idea of The Semantic Web. Then basic description logics is introduced with its parallel in a couple of~formalisms for~building of~ontology. In this work, there is shown, how the ontologies are employed in the semantic annotation process and also there are described some projects that use semantic annotation in a practical way.
